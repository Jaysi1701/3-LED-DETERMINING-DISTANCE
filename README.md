# 3-LED-DETERMINING-DISTANCE
WARNING LEDS DETERRMINING DISTANCE OF THE OBJECT USING DIFFERENT COLOURS CODES (GREEN=DISTANT, YELLOW= MODERATE, RED=TOO CLOSE).


## Components Required
1. Arduino IDE
2. Arduino Uno
3. Ultrasonic Sensor(HC-SR04)
4. LED(X3)
5. Resistor(X3)
